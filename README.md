## 📌 Deskripsi Proyek

Repository ini berisi dokumentasi perjalanan praktikum saya dalam mata kuliah **Jaringan Komputer** di Laboratorium Informatika Telkom University.

Fokus utama repository ini adalah eksplorasi mendalam terhadap arsitektur internet menggunakan pendekatan **Top-Down**, dimulai dari:

- Application Layer  
- Transport Layer  
- Network Layer  
- Physical Layer
